🐕‍🦺 ¿Qué es GitHub?

GitHub es una plataforma basada en la nube que permite almacenar, gestionar y compartir proyectos que utilizan Git. Es como una red social para desarrolladores, donde pueden colaborar en proyectos, hacer revisiones de código y contribuir a repositorios de código abierto.

📌 Analogía perruna: Git es el perrito organizando sus juguetes en diferentes lugares, y GitHub es su casita donde los guarda todos 🏠🐶.

📜 Comandos Básicos de Git 🐾

📝 Instrucción: Agrega una lista con al menos 5 comandos básicos de Git y su función. Usa emojis o referencias a perritos.

  • 🐶 git clone <url> - Como traer un perrito nuevo a casa 🏡.
  • 🐕 git checkout -b <rama> - Crear una nueva ruta para pasear 🛤️.
  • 🦴 git add . - Guardar los juguetes en la mochila 🎒.
  • 🐾 git commit -m "mensaje" - Anotar en el diario las aventuras del día 📖.
  • 🚀 git push origin <rama> - Enviar las fotos del perrito a la nube ☁️.
